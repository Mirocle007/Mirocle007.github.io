6. 布尔运算

在使用 if 语句时, 有时需要满足多个语句或者其组合, 这时涉及到布尔运算.
布尔运算用到布尔运算符,

  • 布尔或操作 or
    • 语法：
      x or y
      注：x 和 y 表示表达式
    • 作用：
      优先返回真值对象
      当 x 为布尔值为 True 时，返回 x, 否则返回 y（返回的是对象）

如:

  • 布尔非操作 not
    • 语法：
      not x
      注：x 表示表达式
    • 作用：
      直接返回 bool 值, 先计算x的 bool 值, 然后取反
